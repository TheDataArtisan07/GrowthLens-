🚀 GrowthLens

GrowthLens is a full-stack Business Intelligence and Predictive Analytics platform that transforms raw e-commerce data into actionable business insights. Built using Flask, Pandas, Plotly, SQLAlchemy, Bootstrap, HTML, CSS, and JavaScript, it helps businesses analyze revenue, customer behavior, retention, customer satisfaction, and future growth opportunities.

📌 Features
📊 Revenue Analytics
Revenue trends and growth analysis
Revenue by category and state
Top-performing products
Average Order Value (AOV)
Automated revenue insights
👥 Customer Analytics
RFM (Recency, Frequency, Monetary) Segmentation
Customer distribution analysis
Purchase frequency analysis
Top customers by revenue
Customer health metrics
🔄 Retention Analytics
